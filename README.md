# tallerlarboc

Web estática del taller mecánico en Santa Lucía, 1 — Arbós (Alt Penedès).

## Para poner el sitio en marcha

1. **Fotos reales** — Sustituir los `photo-placeholder` por fotos del taller, el equipo y los coches.
2. **Teléfono** — Reemplazar `+34...` por el número real en todas las páginas.
3. **Google Maps** — Sustituir el placeholder del mapa en `contact.html` por un embed real.
4. **Formulario** — Conectar el formulario de contacto a un backend (Formspree, Netlify Forms, o similar).
5. **Subir** — Hostear en Netlify o Vercel. Dominio propio. SSL gratis.

## Páginas

- `/` — Inicio
- `services.html` — Servicios
- `diagnostics.html` — Diagnóstico
- `gallery.html` — Galería
- `contact.html` — Contacto
